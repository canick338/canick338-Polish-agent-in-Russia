# Карточная игра "21" (очки)

Простая карточная игра для Данилы на заводе. Игрок играет против дилера, цель - набрать 21 очко или максимально близко к этому, не переборов.

## Правила игры

1. Игрок и дилер получают по 2 карты
2. Одна карта дилера закрыта (рубашкой вверх)
3. Игрок может:
   - **Взять карту** - получить еще одну карту
   - **Остановиться** - закончить ход и передать ход дилеру
4. Если игрок набирает больше 21 очка - перебор, игрок проигрывает
5. После остановки игрока дилер берет карты до 17 очков
6. Если дилер набирает больше 21 - перебор, игрок выигрывает
7. Победитель определяется по наибольшему количеству очков (не более 21)

## Значения карт

- **Туз (A)** = 11 очков (или 1, если перебор)
- **Валет (J), Дама (Q), Король (K)** = 10 очков
- **Остальные карты** = номинал (2-10)

## Использование в сценариях

В файле сценария используйте команду `MINIGAME`:

```
MINIGAME card_game
```

Или альтернативные названия:
- `MINIGAME cards`
- `MINIGAME карты`
- `MINIGAME 21`
- `MINIGAME очки`

## Переменные после игры

После завершения игры устанавливаются следующие переменные:

- `card_game_won` - 1 если игрок выиграл, 0 если проиграл
- `card_game_player_score` - очки игрока
- `card_game_dealer_score` - очки дилера

## Пример использования в сценарии

```
danila: Давай сыграем в карты!
MINIGAME card_game
danila: {IF card_game_won >= 1}Отлично, я выиграл!{ELSE}Блин, проиграл...{ENDIF}
```

## UI элементы

- **PlayerHand** - контейнер для карт игрока (внизу)
- **DealerHand** - контейнер для карт дилера (вверху)
- **PlayerScoreLabel** - очки игрока
- **DealerScoreLabel** - очки дилера
- **StatusLabel** - статус игры (выиграл/проиграл)
- **HitButton** - кнопка "Взять карту"
- **StandButton** - кнопка "Остановиться"
- **NewGameButton** - кнопка "Новая игра" (появляется после окончания)
- **SkipButton** - кнопка "Пропустить игру"
