# Недостающие PNG эмоции для сцены завода

## Для Данилы (godot/Characters/Danila/):

1. **danila_nervous.png** - Нервничает перед начальником
   - Использование: Когда начальник ругается или даёт задания
   - Пример: `danila nervous "Извините, стараюсь."`

2. **danila_determined.png** - Решительный, сосредоточенный
   - Использование: Когда работает на конвейере
   - Пример: `danila determined "Я справлюсь."`

## Для начальника завода (godot/Characters/boss_of_factory/):

1. **boss_of_factory_angry.png** - Злой, ругается
   - Использование: Когда рабочий делает ошибки
   - Пример: `boss_of_factory angry "Ошибки! Переделывай!"`

2. **boss_of_factory_happy.png** - Довольный, улыбается
   - Использование: Когда рабочий хорошо работает
   - Пример: `boss_of_factory happy "Неплохо. Продолжай."`

3. **boss_of_factory_suspicious.png** - Подозрительный
   - Использование: Когда проверяет работу
   - Пример: `boss_of_factory suspicious "Хм, проверю твою работу."`

## Временные решения:

Если эмоций нет, можно использовать существующие:
- `danila worried` вместо `danila nervous`
- `danila serious` вместо `danila determined`
- `boss_of_factory neutral` для всех случаев
