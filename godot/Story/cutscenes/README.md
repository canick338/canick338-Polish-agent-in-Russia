# Система кат-сцен

Этот каталог содержит сценарии с кат-сценами.

## Файлы

- `CUTSCENE_USAGE.md` - Подробные примеры использования
- `test_cutscene.txt` - Тестовый сценарий

## Быстрый старт

1. Создайте видео в формате `.ogv` или `.webm`
2. Поместите в `godot/Cutscenes/`
3. Используйте в сценарии:

```
cutscene my_video
"Текст после видео..."
```

## Больше информации

См. `/godot/CUTSCENE_GUIDE.md` для подробностей о создании видео.
