# Акт 3: Опасность раскрытия

mark start_act3

background polish_home fade_in

"На следующий день. Ваш номер в гостинице."

marek thinking enter left "Информации пока мало. Нужно быть активнее..."

"Внезапно раздается стук в дверь. Настойчивый."

marek surprised "Кто там?"

volkov neutral "Майор Волков, ФСБ. Откройте, пожалуйста."

marek worried "(Чёрт! ФСБ!)"

choice:
	"Открыть спокойно":
		"Вы открываете дверь, стараясь выглядеть спокойно."
		set suspicion_level 5
	"Попытаться спрятать оборудование":
		"Вы быстро прячете оборудование, но это занимает время."
		set suspicion_level 15
	"Открыть сразу же":
		"Вы открываете сразу, не давая повода для подозрений."
		set suspicion_level -5

volkov serious enter right "Добрый день, господин Ковальски."
volkov neutral "Проверка документов у иностранных граждан. Обычная процедура."

marek neutral "Конечно, вот мой паспорт."

"Майор Волков внимательно изучает документы."

volkov thinking "Польский бизнесмен... Интересно."
volkov neutral "И что вас привело в наш маленький город?"

choice:
	"Рассказать легенду подробно":
		marek neutral "Я изучаю возможности для инвестиций в российские регионы."
		marek happy "Обоянь показалась перспективной из-за местного производства."
		volkov suspicious "Перспективной? Обоянь?"
		set suspicion_level 10
	"Ответить коротко":
		marek neutral "Деловая поездка. Изучаю регион."
		volkov neutral "Понятно."
		set suspicion_level 5
	"Попытаться отшутиться":
		marek happy "Устал от больших городов. Решил посмотреть провинцию."
		volkov suspicious "Странный выбор для бизнесмена..."
		set suspicion_level 15

volkov thinking "С кем вы здесь уже познакомились?"

choice:
	"Назвать всех":
		marek neutral "С Анной из администрации, с Димой в баре..."
		volkov neutral "Понятно. Быстро осваиваетесь."
		set suspicion_level 5
	"Сказать что ни с кем":
		marek neutral "Пока ни с кем особо. Только приехал."
		volkov suspicious "Странно... Мне сказали что вас видели в баре..."
		set suspicion_level 20
	"Только с администрацией":
		marek neutral "Только в администрации регистрировался."
		volkov neutral "Хорошо."

volkov serious "Господин Ковальски, у меня к вам личный вопрос..."
volkov thinking "Вы не задавали много вопросов о заводе?"

choice:
	"Отрицать":
		marek neutral "Нет, почему вы спрашиваете?"
		volkov suspicious "Просто проверяю. Это стратегический объект."
		set suspicion_level 10
	"Признаться частично":
		marek neutral "Интересовался, как бизнесмен. Хотел узнать о местной промышленности."
		volkov serious "Понятно. Но знайте - завод под особым контролем."
		set suspicion_level 15
	"Попросить объяснений":
		marek confused "А что не так? Я просто бизнесмен."
		volkov neutral "Ничего не так. Просто предупреждаю."
		set suspicion_level 5

volkov neutral "Что ж, спасибо за ответы. Документы в порядке."
volkov serious "Приятного пребывания в Обояни, господин Ковальски."

"Майор уходит."

marek worried "Это плохо. Очень плохо..."
marek thinking "Он явно что-то подозревает. Нужно решать..."

choice:
	"Продолжить миссию осторожно":
		marek serious "Нет, я не могу сдаться. Буду осторожнее."
		set mission_status continue
		jump act3_continue
	"Подумать о побеге":
		marek worried "Может, пора уезжать? Риск слишком велик..."
		set mission_status retreat
		jump act3_escape
	"Связаться с центром":
		marek thinking "Нужно связаться с центром и доложить ситуацию."
		set mission_status report
		jump act3_contact

mark act3_continue

"Вы решаете продолжить миссию, но быть максимально осторожным."

transition fade_out
